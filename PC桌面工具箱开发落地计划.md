## 一、项目定位

### 核心定位
**DevToolBox** —— 后端开发者专属桌面工具箱（Windows PC客户端）

### 核心优势
- ✅ **专属差异化**：垂直服务 Java 开发者日常高频工具，精准刚需
- ✅ **技术零门槛**：全程 Java 17 + JavaFX 17，100% 原生技术栈
- ✅ **上线极简**：无需域名、服务器、备案、应用商店审核
- ✅ **AI全程赋能**：界面布局、样式、BUG修复、打包脚本全部 AI 生成
- ✅ **可迭代可沉淀**：第一版快速上线，后续无限加功能

### 产品特性
- 🔒 **纯本地运算**：无网络请求、无数据上传、零合规风险
- 🎨 **深色主题**：专为程序员设计的护眼配色
- 📦 **一键打包**：自动生成 Windows 安装包（含 JRE，用户免装 Java）
- 🚀 **高性能**：原生 JavaFX，流畅响应

## 二、技术栈

### 已选定技术
| 项目 | 技术 | 版本 | 说明 |
|------|------|------|------|
| 开发语言 | Java | 17 | LTS 长期支持版 |
| 桌面框架 | JavaFX | 17.0.8 | 官方原生桌面框架 |
| 构建工具 | Maven | 3.9+ | 依赖管理 |
| UI 布局 | FXML | - | 声明式界面定义 |
| 打包工具 | jpackage | JDK 17+ | 一键生成 Windows exe |
| 日志框架 | logback | 1.4.11 | 日志记录 |
| JSON 处理 | Jackson | 2.16.1 | JSON 解析/格式化 |

### 为什么选 JavaFX？
- ✅ 完全 Java 语法，零新语言学习成本
- ✅ 官方支持，长期维护
- ✅ 现代化 UI，支持 CSS 样式
- ✅ 跨平台（Windows/Mac/Linux）
- ❌ 不选 Swing：界面老旧
- ❌ 不选 Electron：体积大、脱离 Java
- ❌ 不选 C# Winform：需要学新语言
